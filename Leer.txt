NOTAS:

1. Instalar bootstrap en angular:
	>> npm i bootstrap --save

2. Configurar bootstrap en angular:
	-> Abrir el archivo "angular.json".
	-> Buscar la propiedad "styles".
	-> Aumentar el siguiente codigo:
        "styles": [
            "src/styles.css",
            "node_modules/bootstrap/dist/css/bootstrap.min.css"
        ]
